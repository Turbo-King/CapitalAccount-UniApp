# CapitalAccount-UniApp

### 🌈 项目概述

个人资金账户管理系统（前端）

⚠️  注意：
此项目为前端UniApp项目。


### 🔨 项目结构

```
CapitalAccount-UniApp
│
├─components 公共组件模块
│
├─pages 页面模块
│  
├─static 静态资源模块
│  
├─util 公共工具模块
│  
├─uview-ui uview页面资源模块
│  
├─App.vue 全局样式变量配置文件
│  
├─main.js 全局js配置文件
│  
├─maniferst.json 程序发布基础配置文件
│  
├─pages.json 全局页面配置文件
│  
└─uni.scss 全局样式文件

```


### ⚡ 快速启动

1. clone仓库项目
2. 将项目导入HBuilderX
3. 启动项目
